## Technologies
Project is created with:
* ReactJS 17.0.2
* New York Times API
	
## Setup
To run this project, install it locally using npm:

```
$ cd assesment
$ cd nytimesapp
$ npm start
```

## Unit Testing
To do unit testing , using npm:

```
$ cd assesment
$ cd nytimesapp
$ npm run test
```


## Coverage Report
To do a coverage report , using npm:

```
$ cd assesment
$ cd nytimesapp
$ npm run test -- --coverage --watchAll=false

```
